# tech-interview  
A practical, no nonsense collection of tools, prompts, checklists, and resources for evaluating tech candidates.  
This repo is built for hiring managers, interviewers, and senior engineers who want clear signal, honest assessments, and zero fluff.

## What this repo contains
This repository focuses on practical and brutally honest evaluation tools used in real tech hiring. It includes:

### 1. CV and Resume Evaluation Prompts
High clarity, critical evaluation frameworks designed to surface:
- Real skills vs claimed skills  
- Weak fundamentals  
- Technical depth  
- Communication quality  
- Portfolio credibility  
- Cultural fit signals  
- Motivation honesty  
- Immediate dealbreakers  

The prompts are tuned for junior to mid-level tech roles and emphasize clean formatting, professional presentation, and evidence based assessment.

### 2. Interview Question Resources (optional section for future expansion)
A structured collection of interview materials such as:
- Technical screening questions  
- Problem solving exercises  
- System design warmups  
- Behavioral prompts that expose real ownership and thinking  
- Red flag checklists  
- Practical take-home guidelines  

These materials aim to help teams run consistent, high signal interviews that avoid guesswork.

### 3. Evaluation Rubrics
Simple, fast scoring templates for:
- Technical ability  
- Communication skill  
- Coachability  
- Problem solving approach  
- Team fit  
- Growth potential  

Each rubric encourages concise notes and objective scoring.

### 4. Improvement Plans and Feedback Templates
Short, honest improvement plans candidates can use to fix weak CVs or portfolios.  
Also includes concise templates for giving rejection feedback without sugarcoating.

## Why this repo exists
Most interview resources online are bloated, generic, and overly positive.  
This repo focuses on clarity, evidence, and actionable insights.  
The goal is to help interviewers make better decisions and help candidates understand exactly what they need to improve.
